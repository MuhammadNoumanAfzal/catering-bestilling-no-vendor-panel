# catering-bestilling-no-vendor-panel
Marketplace platform for catering vendors to manage menus, orders, bookings, and customer requests online.

## Auth Environment

- `VITE_GRAPHQL_API_URL`: GraphQL endpoint for auth requests
- `VITE_AUTH_ROLE=vendor`: required role for this vendor panel
- `VITE_CLOUDINARY_CLOUD_NAME`: Cloudinary cloud name for direct menu image uploads
- `VITE_CLOUDINARY_UPLOAD_PRESET`: unsigned upload preset for direct menu image uploads
